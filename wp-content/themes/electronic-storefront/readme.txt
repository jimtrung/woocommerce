== Description ==

Electronic Storefront is a modern, responsive WooCommerce WordPress theme designed to create professional online shops and storefronts. Perfect for fashion boutiques, lifestyle brands, homeware sellers, and marketplaces, it combines sleek design with powerful WooCommerce features like product wishlists, comparison tools, and variation swatches. Fully compatible with Elementor, Divi, Visual Composer, and Gutenberg, it offers effortless customization, while its SEO-friendly, mobile-responsive, and fast-loading structure ensures a smooth shopping experience across all devices. With RTL support, flexible layouts, and conversion-focused styling, Electronic Storefront is the ideal shop theme for building a standout WooCommerce storefront.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
